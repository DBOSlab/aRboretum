# Package index

## Main workflow

Core functions for preparing data and generating outputs.

- [`arboretum_data()`](https://DBOSlab.github.io/aRboretum/reference/arboretum_data.md)
  : Extract and compile flora data from multiple taxonomic databases
- [`arboretum_audios()`](https://DBOSlab.github.io/aRboretum/reference/arboretum_audios.md)
  : Create folder structure and phrase files for personal audio
  recordings
- [`arboretum_photos()`](https://DBOSlab.github.io/aRboretum/reference/arboretum_photos.md)
  : Create folder structure for personal species photos
- [`arboretum_labels()`](https://DBOSlab.github.io/aRboretum/reference/arboretum_labels.md)
  : Generate multilingual HTML labels for plant collections
- [`arboretum_minisite()`](https://DBOSlab.github.io/aRboretum/reference/arboretum_minisite.md)
  : Build a multilingual minisite index for generated species labels
- [`arboretum_qrcodes()`](https://DBOSlab.github.io/aRboretum/reference/arboretum_qrcodes.md)
  : Generate printable QR code labels for plant collection specimens

## Package data

Internal datasets used by the package workflow.

- [`botregions`](https://DBOSlab.github.io/aRboretum/reference/botregions.md)
  : Countries and associated classification of botanical divisions
- [`WGSRPD`](https://DBOSlab.github.io/aRboretum/reference/WGSRPD.md) :
  World Geographical Scheme for Recording Plant Distributions - Spatial
  Map
